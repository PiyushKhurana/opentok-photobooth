# opentok-photobooth

## Set up frontend with Vue.js

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```

## To run our Python Backend

```
gunicorn -b 0.0.0.0:80 server:app
```
